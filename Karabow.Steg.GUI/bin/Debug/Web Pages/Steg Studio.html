Steg Studio allows you to hide a document in an image.

To hide a file in an image:
  • Go to Task>Encryption, or press Ctrl+E
  • A tab "Encryption Page" opens
  • Select a file to hide, or an entire folder, or a type of file.
    For instance, you can type "C:\path\to\folder\*.html" to select all ".html" files in the folder "C:\path\to\folder"
  • Click on the big square box to select the image to use for hiding the file.
    Don't worry, the file won't be hidden in this image but in an identical image that would be generated by the software.
  • Decide if the original file you are hiding should be deleted.
  • Specify your password. You must use this password when recovering the file from the image.
  • Click the "Encrypt" button.
    A new image containing your file is created in the same folder as the file you are hiding, with an auto-generated name. You will, however, be asked to specify another name for the image if the auto-generated name conflicts with the name of a file in that folder.
To recover a file from an image:
  • Go to Task>Decryption, or press Ctrl+D
  • A tab "Decryption Page" opens
  • Select the image (or images) from which the files should be recovered.
  • Enter your password, the same password you used while hiding the original file in this image.
  • Decide if the image from which you are recovering the file should be deleted.
  • Click the "Decrypt" button.
